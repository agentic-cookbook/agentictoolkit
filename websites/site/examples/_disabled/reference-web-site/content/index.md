# Welcome

This is a tiny demo knowledge base running inside the toolkit's examples
shell. The content here is plain markdown — the
[`reference-web-site` package](https://github.com/agentic-cookbook/agentic-web-toolkit)
turns it into a navigable SPA.

## What you're seeing

- A configurable header (title, GitHub link, search, theme toggle)
- A nav tree built from the directory structure of the markdown files
- Per-page tables of contents
- Fuzzy search (try **⌘K**)
